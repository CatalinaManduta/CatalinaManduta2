What is Frequent Pattern Mining?

Frequent pattern mining in machine learning is a process used to find patterns that appear often within a dataset. It helps identify items that frequently appear together, like bread and butter, which are commonly bought together. It's about discovering which combinations or sequences of things show up regularly in a set of data, useful in understanding relationships such as gene co-expression patterns, risk factor identification, or health behavior patterns.

Sets

Before diving into the subject, here are a few key concepts:

Definitions

  • Extensional Definitions: Helpful when you have a limited set of elements. Example: S = {sparrow, eagle, pigeon, robin}.
  • Intensional Definitions: Describes larger or infinite sets by rule. Example: S = {x ∈ Birds | x’s wingspan > 1 meter}.

Set Properties

  • Uniqueness: Each element in a set is unique (e.g., influenza cases).
  • Well-Definedness: A set must be well-defined, like D = {individuals with fasting blood sugar > 126 mg/dL}.
  • Order Irrelevance: The order of elements in a set doesn't matter ({A, B, C} = {C, B, A}).
  • Finite vs. Infinite: Finite sets have a countable number, while infinite sets don’t (e.g., future humans at risk for a novel virus).
  • Subset: All elements of a subset are part of a larger set (T ⊆ H).
  • Empty Set (∅): A set with no elements. For instance, no individuals with a mutation means ∅.
  • Cardinality: The number of elements in a set. Example: T = {John, Anna, Maria}, cardinality = 10,000.

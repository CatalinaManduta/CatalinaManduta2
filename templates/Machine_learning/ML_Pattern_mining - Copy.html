Frequent Pattern Mining

What is Frequent Pattern Mining?

Frequent pattern mining in machine learning is a process used to find patterns that appear often within a dataset. Imagine you have a large collection of data, like a list of things people buy from a store. Frequent pattern mining helps identify items that frequently appear together, like bread and butter, which are commonly bought together. In simple terms, it's about discovering which combinations or sequences of things show up regularly in a set of data. This is useful in understanding relationships between different items or events, such as in gene co-expression patterns, risk factor identification, or health behavior patterns.

Sets

Before diving into the subject, there are a few important concepts to introduce:

Definitions

  • Extensional Definitions: These are helpful when you have a limited, manageable number of elements in a biological set, and you can explicitly list them. For example, you might define the set of bird species in a specific park as: S = {sparrow, eagle, pigeon, robin}.
  • Intensional Definitions: These are useful for describing larger or infinite sets based on a rule or condition. For instance, to define all bird species with a wingspan greater than 1 meter: S = {x ∈ Birds | x’s wingspan > 1 meter}.

Set Properties

  • Uniqueness: In a set, each element is unique. This prevents confusion and ensures clarity in data analysis. For example, if you're tracking influenza cases, each patient appears only once in the set.
  • Well-Definedness: A set must be well-defined, meaning that for any element, you can clearly determine whether it belongs to the set. For example, D = {all individuals with a fasting blood sugar level > 126 mg/dL}.
  • Order Irrelevance: The order of elements in a set doesn't matter. For instance, {A, B, C} is the same as {C, B, A}.
  • Finite vs. Infinite Sets: A finite set has a specific, countable number of elements, while an infinite set doesn't. For example, C = {100,000 infected individuals} is finite, but S = {all future humans at risk for a novel virus} is infinite.
  • Subset: A subset contains elements that are all part of a larger set. For example, T ⊆ H means every element in T is also in H.
  • Empty Set (∅): An empty set contains no elements. For instance, if no individuals in a study have a particular mutation, the set of affected individuals is ∅.
  • Cardinality: Cardinality is the number of elements in a set. For example, if T = {John, Anna, Maria, ...}, the cardinality of the set would be the total number of individuals, say 10,000.

Classical genetics focuses on how traits (phenotypes) are passed down and how they work, without needing to know the specific details about the genes, proteins, or other molecules involved. It's like figuring out how something works by observing what you can see (like physical traits or behaviors) without needing to understand the inner workings or the exact parts that make it happen.

Phenotype

Phenotype is what you can see or measure in an organism, like its eye color, height, or how it behaves. It's how the genes and environment together make the organism look or act.
